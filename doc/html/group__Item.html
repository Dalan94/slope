Detailed Description

The base class for item objects.

Author
Elvis Teixeira
Date
18 Jan 2015

Everithing that can be drawn in a slope_figure_t, including data representations, chart axis, legends and text labels, are items. To be drawn, items must lie in the item list of metrics that that lie in the metrics list of a figure.
