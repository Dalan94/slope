Detailed Description

Figure is the object that holds any of slope's drawables.

Author
Elvis Teixeira
Date
18 Jan 2015

Every time one wants to produce a figure one will instantiate a slope_figure_t, add one or more slope_metrics_t to it and then place the slope_item_t's in the metrics. All that remains to do is draw the figure to any of cairo's backends, such as a GtkWidget a PNG file or any other.
